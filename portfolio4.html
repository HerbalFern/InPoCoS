Hardware

What is Computer Hardware?

Hardware

  • Hardware is basically everything comprising of a system that is not part of software (Clements, 2006).
  • Hardware is the physical component of the computer system.

Computer hardware contains the Central Processing Unit, Functional computer parts, and other Peripheral devices (Frankenfeld, 1993).


Notable Computer Hardware

The Central Processing Unit

  • The CPU is known as the “brain” of the computer
  • According to Clements (2006), the CPU is both the control unit and logic unit of the system.
    • It is the control unit of the computer as it gives out instructions for the operations of a computer.
    • It is the logic unit because it performs the arithmetic functions under the direction of the control unit.

Random-Access Memory (RAM)

  • RAM, or random-access memory, serves as the main memory in a PC. It temporarily holds all active programs and data, making them easily accessible to the CPU and other system components.
  • When you run a program on your PC, a copy of it is loaded into RAM from secondary storage, typically the hard drive, allowing quick access (Gilster, 2001).

Motherboard

  • The motherboard consolidates all the PC's essential system components onto a single printed circuit board (PCB). In this single-board design, all the electronic circuitry that manages and connects the PC's operations is located on the motherboard (Gilster, 2001).

Graphical Processing Unit

  • Graphics Card are utilized for the graphical output of the computer. The GPU is essential for running programs that are graphics intensive and is considered the heart of the computer (Gilster, 2001).

Read-Only Memory

  • Read-Only Memory (ROM) is a type of computer memory that stores data which can be read but not modified. This is where the user stores their data. It is non-volatile, meaning it retains its contents even when the device is powered off (Clements, 2006).

Peripheral Devices

  • Storage Devices are devices which store data within the computer. It can be external or internal
  • Input/Output Devices are devices that input data to the computer and display output to the users (Frankenfield, 1993).

Software

What is Software?

Software

   Software is a set of instructions, data, or programs used to operate computers and execute specific tasks. It is the opposite of hardware, which describes the physical aspects of a computer. Software is essential for a computer to function, as it gives the hardware the ability to perform its designed tasks (Sommerville, 2015; Hunt & Thomas, 1999).


Types of Software

    System Software

  • Operating Systems
    • Manage computer hardware and software resources (e.g., Windows, macOS, Linux) (Sommerville, 2015).
  • Utility Programs
    • Perform maintenance tasks (e.g., antivirus software, disk cleanup tools) (Sommerville, 2015)

    Application Software

  • Productivity Software
    • Helps users accomplish specific tasks (e.g., Microsoft Office, Adobe Photoshop) (Hunt & Thomas, 1999).
  • Business Software
    • Assists in business processes (e.g., ERP systems, CRM software) (Hunt & Thomas, 1999).
  • Media Players and Editors
    • Handle multimedia files (e.g., VLC Media Player, Adobe Premiere Pro) (Hunt & Thomas, 1999).

    Programming Software

  • Compilers
    • Convert code written in a programming language into machine code (e.g., GCC, Visual Studio) (Sommerville, 2015).
  • Debuggers
    • Software that helps identify and fix errors in code (Sommerville, 2015).
  • Integrated Development Environments (IDEs)
    • Provide comprehensive facilities to programmers (e.g., Eclipse, IntelliJ IDEA) (Hunt & Thomas, 1999).

    Embedded Software

  • Software written to control machines or devices that are not typically thought of as computers, such as cars, phones, and appliances (Sommerville, 2015).

    Network Software

  • Manages network communication (e.g., network operating systems, network management software) (Sommerville, 2015).

    Driver Software

  • Allows the operating system and other software to interact with hardware devices (e.g., printer drivers, graphic card drivers) (Sommerville, 2015)

Implementations in different institutions in the Community

Business

Businesses within the community can apply hardware and software technologies to give expand their services or make services for the community. Such implementations are linking community markets through a network for a more convenient solution to selling. There are more implementations of software and hardware in business, but online marketing and networking is definitely one of them. (Singh et al., 2023)

Education

Software is actively being utilized in the current information systems of schools. In the Philippines, most universities now have official information systems which help manage the student’s school related activities and transactions (Grepon et al., 2021). Hardware such as computers are also being utilized for enhanced student learning and enhancing the technological proficiency of students (Timotheou et al., 2023).

Healthcare

eHealth technologies are being utilized in hospitals, from managing patient data to optimizing hospital process for efficient operations which is highly beneficial for the community (Nilsen et al., 2020). Software and hardware is heavily utilized in improving the overall experiences in hospitals and provide much better care for patients (Kim et al., 2017). Without hardware or software, we would not be able to have hospitals that can cater as much patients as we do now.

Government

Governments can use hardware and software to improve efficiency, save money, and provide better services to the public. Things such as automation of transaction records, easier management of citizens and anti-cybercrime methods are now possible through newer software and hardware. Technology can improve efficiency, citizen involvement, and government accountability, but also addresses obstacles like the digital divide and data security within the community. (Latupeirissa et al., 2024)
